## write

博客后台管理系统

* Golang 1.16+

前端目录位于[web/app](https://github.com/moonprism/blog/tree/master/write/web/app)

### Channel(php)

* `gRPC` 与后台程序之间的通信，获取 Golang 全文检索库 riot 的查询结果
* `CAS` Golang 后台作为认证服务器
* `redis` 用于邮件推送队列等
