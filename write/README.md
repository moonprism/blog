## write

博客的后台管理系统，基于golang和vue。前端目录位于 `web/`

#### 运行

查看构建帮助：
```
make help
```

指定dev配置运行：
```sh
go run main.go --env dev
```

> 运行dev开发环境，在write下必须存在`log`与`static`目录
